# Scripts

- `build_macos.sh` - configure and build Release with Ninja (requires `JUCE_SOURCE_DIR`).
- `install_macos.sh` - copy AU/VST3 bundles into user plugin folders.
- `dev_loop.sh` - configure/build Debug, install, and watch for changes (uses `fswatch` if available).
- `open_xcode.sh` - generate and open the Xcode project.
