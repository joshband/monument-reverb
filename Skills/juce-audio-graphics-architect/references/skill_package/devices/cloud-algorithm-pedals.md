# Cloud Algorithm Pedals and Related Gear (informal notes)

This list captures ambient reverb or texture pedals that align with the "cloud algorithm" idea.
Notes are informal and do not include citations.

## Pedals
- Strymon Cloudburst Ambient Reverb Pedal: ambient reverb based on the Cloud lineage, with Ensemble mode for evolving pads.
- Tomkat Cloudy V2 Digital Multi-Effect Pedal: boutique digital pedal focused on granular and textural ambience.
- Maneco Labs Small Nimbus (Cloud Module Pedal): pedal implementation of Mutable Instruments Clouds in a guitar format.
- Hologram Electronics Microcosm Granular Pedal: granular and looping textures for evolving soundscapes.
- Mooer A7 Ambience Reverb Pedal: budget ambient reverb with multiple algorithms.
- Caline CP-508 Wonderland Ambient Reverb Pedal: simple ambient reverb with shimmer and decay control.
- Diamond Dark Cloud Midnight Sparkle Delay Pedal: hybrid delay that can blend warm repeats with airy ambience.
- Chaos Audio Stratus Multi-Effects Pedal: multi-effect platform with ambience and delay blocks.

## Usage ideas
- Map "cloud" behaviors to long decay, dense diffusion, shimmer, or granular pitch scatter.
- Use these as mood references for UI language, presets, or demo patches.
