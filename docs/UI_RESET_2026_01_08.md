# Monument Reverb UI Reset - 2026-01-08

## Summary

The Monument Reverb UI has been reset from a complex photorealistic system to a simple, clean JUCE-based approach. This change was made to provide a clean slate for learning and focus on functionality over aesthetics.

## What Changed

### Archived (moved to `archive/ui-v1-2026-01-08/`)

**UI Components:**
- CelestialKnob (layered RGBA with blend modes)
- PhotorealisticKnob (filmstrip-based)
- LayerCompositor (blend mode utility)
- MacroVisualOverlay (reactive particles)
- ModMatrixPanel (4×15 grid)
- HeaderBar, CollapsiblePanel, TimelineComponent
- EnhancedBackgroundComponent (animated wisps)
- MonumentControl, ChamberWallControl, LayeredKnob, HeroKnob
- StoneBackgroundComponent

**Assets:**
- Celestial knob layers (base, veins, glow)
- Processing scripts
- Visual profiles and macro hints JSON

**Build System:**
- Binary asset embedding (BinaryData)
- Complex UI component linking

### New Implementation

**Simple JUCE Components:**
- 10 rotary sliders (Slider::RotaryVerticalDrag)
- Labels for each parameter
- Dark background (0xff1a1a1a)
- 2×5 grid layout
- 800×600 window size

**Parameters:**
1. Material
2. Topology
3. Viscosity
4. Evolution
5. Chaos (chaosIntensity)
6. Elasticity (elasticityDecay)
7. Patina
8. Abyss
9. Corona
10. Breath
