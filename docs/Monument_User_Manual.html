Monument reverb

Monument User Manual

Monument is an architectural reverb designed for impossible-scale spaces. Think weight, depth, and slow-moving density rather than faithful room realism. This guide is practical and playful so you can move quickly and still feel the monument shift around you.

AU + VST3 macOS Apple Silicon FDN core Warp + Drift Freeze safe

Quick start

1. Build and install

Use the scripts for the shortest path to a running plugin.

./scripts/build_macos.sh
./scripts/install_macos.sh

2. Load and listen

Insert Monument on a send or insert, raise Mix, then lengthen Time to reveal the tail.

Tip: Monument is not subtle by default. Start at Mix 20-30, Time around 0.6, Density around 0.5.

Signal flow

Monument is modular. Each stage is real-time safe and designed for large-scale ambience.

Foundation: input prep Pillars: early reflections Chambers: FDN core Weathering: motion Buttress: safety Facade: image and tone

Control compass

All controls are normalized [0, 1] unless noted. Mix is [0, 100].

ControlIntent
TimeOverall decay length
MassWeight and darkness
DensityReflection complexity
BloomLate swell after the source stops
AirUpper-band openness
WidthStereo reach of the wet field
GravitySpectral tilt and decay weighting
WarpSlow topology bending
DriftMicro motion in the tail
MixWet/dry balance
FreezeInfinite hold of the current state
Freeze holds the late field with unity feedback and no new input. Warp and Drift pause during Freeze and ramp back in smoothly on release.

Freeze transitions

Freeze crossfades between the live wet path and the frozen wet path to avoid clicks.

stateDiagram-v2
  [*] --> Live
  Live --> Freezing: Freeze on
  Freezing --> Frozen: outputBlend -> 0
  Frozen --> Releasing: Freeze off
  Releasing --> Live: outputBlend -> 1

Build and install notes

CMake build

cmake -S . -B build -G Xcode -DCMAKE_OSX_ARCHITECTURES=arm64
cmake --build build --config Release

Tests

ctest --test-dir build -C Release
Artifacts land in build/Monument_artefacts. Install using scripts/install_macos.sh.

Troubleshooting

No reverb sound

Raise Mix and ensure Freeze is off. Check that Density is above 0.2 for a continuous tail.

Tail is too long

Lower Time or increase Mass and Gravity to reduce high band sustain.
