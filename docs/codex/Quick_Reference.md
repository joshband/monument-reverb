# Codex Quick Reference

## Prompt template
"Goal: ...
Constraints: ...
Context: ...
Output: ..."

## Common requests
- "Make a plan and wait for approval."
- "Implement the change and run tests."
- "Give me a code review with risks and missing tests."
- "Summarize changes and list modified files."

## When asking for changes
- Name the exact file or folder when possible.
- Clarify the desired behavior and edge cases.
- Specify required formatting or style guides.

## When asking for tests
- Name the command to run.
- Specify build config (Debug or Release).
- Ask for a summary of failures if any.
