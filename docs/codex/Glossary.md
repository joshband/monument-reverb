# Glossary

- Approval: A prompt that asks you to allow Codex to run a command or access resources.
- Sandbox: Restrictions on file system or network access.
- Skill: A bundle of instructions and assets for specialized tasks.
- Session: A single working conversation with Codex.
- Workspace: The folder Codex is allowed to read and write.
- Tool: A capability Codex can call (shell command, file edit, etc.).
